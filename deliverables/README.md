이 PDF들은 같은 폴더의 HTML에서 생성됩니다. 수정 후 재생성:
```bash
# 예) 기획서
"/Applications/Google Chrome.app/Contents/MacOS/Google Chrome" --headless=new \
  --no-pdf-header-footer --print-to-pdf=deliverables/Tradar_제품및서비스개발부문_기획서.pdf \
  "file://$PWD/deliverables/기획서.html"
```
